Solution Options

Click here to view/hide more information
Click here for Resources & Guidance

Solution options are ways to address a decision problem, e.g., different strategies for supporting below-grade readers.

Decision you are working on:
Logged in as:
I. List below each program/strategy/option you think should be considered as a way to address the issue about which you are making a decision. Add the source of the idea.
II. When you are done, click here: Save and Back to Flowchart
 

Optional Steps:
III. Add or view any relevant information, web sites or documents for each option using the “Solution Option Details” buttons.
IV. If you want to share this page with other people without them logging into DecisionMaker, you can create a public link to embed in an email or other communication: Create a Public Link
V. The Project Administrator can temporarily remove any of the options to take them out of the consideration set by clicking on “Put this away for now” under each option. You can retrieve them any time from here: Archived Solution Options
VI. If the Project Administrator wants to invite stakeholders to suggest Solution Options or review who has already been invited, click here: Save and Add/View Stakeholders

If you need to enter additional Solution Options, more empty boxes will appear after you click on “Save and Back to Flowchart”.
