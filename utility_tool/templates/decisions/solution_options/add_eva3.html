Evaluation Criteria


Use this page to select/enter factors (“Evaluation Criteria”) that are important to you and other stakeholders and that you will use to evaluate each Solution Option.


You can select/enter as many criteria as you think it will be feasible to evaluate each option against. The average number used is 6-7.


Example: Evaluation Criteria you might want to use for a decision about a new elementary school literacy curriculum might include:


Impact on reading comprehension

Fit with local context - Fit with local cultural values

Support from stakeholders - Teacher preference/buy-in/support

Feasibility of implementation - Fit with school calendar/schedule.

Step 1:

You can select Evaluation Criteria from the suggested list below or enter your own in the empty text boxes underneath. When you check one of the suggested “overarching” criteria, a second list of more specific criteria will appear and it is these that you will use to evaluate each of the Solution Options you are considering. You can view the entire list here.

Note: DecisionMaker assumes that costs are always a factor in decision-making. In a later step, you will be asked to enter estimated costs of implementing each Solution Option so, while you may have screened out any options that clearly exceed your budget, should not specify costs as an evaluation criterion here.
